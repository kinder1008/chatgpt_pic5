# ChatGPT 圖像提示詞工具 - 對外發布版

這個資料夾可直接上傳到靜態網站主機使用。

## 需要上傳的檔案

- index.html
- styles.css
- app.js
- prompts-data.js

## 建議發布方式

### 方式 1：Netlify Drop

1. 打開 https://app.netlify.com/drop
2. 將整個「04_對外發布版」資料夾拖進網頁
3. Netlify 會產生一個公開網址
4. 把網址分享給外部使用者

### 方式 2：GitHub Pages

1. 建立一個 GitHub repository
2. 上傳本資料夾內的 4 個檔案
3. 到 Settings > Pages
4. Source 選擇 main branch / root
5. 儲存後等待 GitHub 產生公開網址

### 方式 3：一般網站空間

將本資料夾內的 4 個檔案上傳到網站根目錄或子資料夾即可。

## 隱私說明

圖片上傳只會在使用者自己的瀏覽器中預覽，不會上傳到伺服器。
這個網站目前沒有後端資料庫，也不會儲存使用者輸入的內容。

## 注意事項

如果要讓網址更正式，可以購買網域，並綁定到 Netlify、Vercel、GitHub Pages 或自己的主機。
